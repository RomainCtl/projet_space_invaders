# Projet Java fin de Première année

> Projet build with Java 11 (openjdk 11.0.3) et Swing

## Compile & Run

> clear compiled objects : `make clean`

> compilation : `make build`

> execution : `make run`

> Do `make` to do all (clean, build and run)
